# PWA Mobile - Notas de Jogos de Tabuleiro

Versão mobile-first do app de avaliação de jogos de tabuleiro.

## Recursos
- Layout otimizado para celular.
- Avaliação por cartões.
- Sliders de nota de 1 a 10.
- Pesos automáticos por categoria.
- Descrição completa de cada nota por categoria.
- Bônus emocional de até +0,5.
- Nota final limitada a 10.
- Campo de comentários.
- Resumo da avaliação.
- Salvamento local da última avaliação.
- Manifest e service worker para instalação como PWA.

## Como testar localmente
```bash
python -m http.server 8000
```
Depois acesse `http://localhost:8000`.

## Como publicar no GitHub Pages
Envie todos os arquivos para um repositório e ative em `Settings > Pages`.
